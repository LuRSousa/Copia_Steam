## Cópia da Interface Steam

Este projeto é um estudo de interface (UI Clone) da plataforma Steam. O objetivo foi replicar a identidade visual da loja, focando na organização de componentes complexos, manipulação de listas dinâmicas com JavaScript e a aplicação de estilização avançada para manter a fidelidade ao design original.

### Tecnologias
- **JavaScript** (Lógica de exibição e funções)
- **HTML5 & CSS3**
- **Flexbox & Grid Layout** (Para a estrutura da vitrine)

### Status do Projeto
**Interrompido** (Fev/24)
*Este projeto serviu como uma etapa importante de aprendizado em estrutura de layout, sendo pausado para o foco em novos projetos com tecnologias mais recentes.*
